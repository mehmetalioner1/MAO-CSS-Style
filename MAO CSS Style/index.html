You are in Positioning

Positioning

Static
Relative
Absolute
Fixed
Sticky

Static: It remains at the point where it is added in the document flow. Its position can be influenced by properties like display, margin, and padding.

Relative: Positioned relative to its normal position in the document flow. It can be moved using top, right, bottom, and left properties, without affecting the positions of surrounding elements.

Absolute: Positioned relative to the nearest positioned ancestor (or the initial containing block if no positioned ancestor is found). It's taken out of the normal document flow. It can be precisely placed using top, right, bottom, and left properties.

Fixed: Positioned relative to the viewport or the initial containing block. It remains fixed in its position even when the page is scrolled. Commonly used for elements like headers or navigation bars that should always be visible. Its position can be adjusted using top, right, bottom, and left.

Sticky: Initially positioned according to the normal flow of the document, but behaves like fixed within a specified scrollable area. It remains in its normal position until it reaches a specified point (e.g., top of the viewport) as the page is scrolled. This behavior is controlled by setting top, right, bottom, or left properties to define the sticky point.
